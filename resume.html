# TJ Blechman Portfolio A cinematic, responsive personal portfolio for TJ Blechman, a Computer Science student and product-focused engineer. The site presents selected client, software, and engineering work through immersive case-study layouts rather than a conventional grid of project cards. ## Features - Responsive single-page portfolio experience - Animated signal-network canvas and scroll-driven reveals - Full-width project case studies - Faithful visual previews of featured work - Interactive `TJ_OS` terminal - Academic and builder timeline - Dedicated printable résumé page - Mobile navigation and reduced-motion support - Custom favicon, manifest, and 404 page ## Built With - Semantic HTML5 - Modern CSS - Vanilla JavaScript - Canvas API The project is intentionally dependency-free and requires no framework, package manager, or build process. ## Project Structure ```text . ├── index.html # Main portfolio ├── styles.css # Layout, design system, and responsive styling ├── script.js # Interactions and animations ├── resume.html # Printable résumé ├── assets/ # Project imagery ├── favicon.svg # Site icon ├── site.webmanifest # Web app metadata ├── 404.html # Custom not-found page └── .nojekyll # GitHub Pages configuration ``` ## Design Direction The portfolio is designed to communicate technical ability, product judgment, and attention to detail. Motion and visual effects are used deliberately to support the content without compromising clarity, accessibility, or performance. ## Deployment The site is hosted as a static website with GitHub Pages. ## License © 2026 TJ Blechman.
